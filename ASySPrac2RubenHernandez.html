  • Suponga que existe un vector x en el espacio de trabajo de MatLab que corresponde a una señal en tiempo discreto de duración finita.
  • Escriba una función de MatLab que cuando se ingrese el vector x calcule la energía de x[n].
  • Escriba una función de MatLab que cuando se ingrese el vector x calcule la potencia de x[n]. Asuma que x[n] es periódica y que el vector x contiene datos para un número entero de periodos de x[n]
Para la energia de una señal finita
n1= x1 %%El usuario ingresa desde donde empieza la señal discreta
n2= x2 %%El usuario ingresa desde donde termina la señal discreta
n = n1:n2; %% el programa hace un vector desde los puntos donde se hara la energia de la señal
x=@(n) ; %% Se ingresa la funcion anonima que se quiera encontrar su energia
Ex= sum(x(n).^2)%% Se calcula la señal de energia que se quiera
stem(n,x(n)) %% Se hace un ploteo de nuestra señal discreta
stem(n,x(n).^2) %% Se hace un ploteo de la señal discreta elevada al cuadrado
Para la potencia de una señal periodica
N= N0; %% Se anota el periodo para nuestra señal
r=0:N;%% se hace un vector desde cero hasta N0-1
x=@(r) ---; %% se pone la función
Px= (1/N)*sum(x(r).^2)%% se calcula la potencia de la señal
